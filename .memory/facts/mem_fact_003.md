Two persistence mechanisms: (1) SQLite at `data/issues.db` for the issue tracker — managed via `app/db.py`. (2) NumPy arrays at `data/search_embeddings.npy` + `data/search_chunks.json` for semantic search — both gitignored and regenerated by `app/search.py`.
